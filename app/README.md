# MapTiler App

This directory contains everything related to the Maptiler app that can be built and hosted on a server.

## Setup

- `npm install`
- For development with hot reloading: `npm run dev`
- For a production build: `npm run build`
- If you want to test the build, go to the dist (`cd dist`) folder and start e.g. a python server `python3 -m http.server`
